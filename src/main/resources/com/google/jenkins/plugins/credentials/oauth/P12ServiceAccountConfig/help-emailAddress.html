This is the E-Mail address associated with the service account credential in the Google Developer Console.

To create a new service account:

  • Select your project from the Project list.
  • Navigate to "APIs & Auth" and then "Credentials"
  • Click the "Create new Client ID" button
  • Select "Service account" and click "Create Client ID".
  • A dialog should pop-up to download the private key file for the new service account, save this in a secure location. This file contains a Private key in json format.
  • After clicking, "Okay, got it", you should see a "Service Account" section and a Email address field; This is the E-Mail address you must input here.
